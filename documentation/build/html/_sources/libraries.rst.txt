pyacs core libraries
====================

* :doc:`./pyacs.lib.astrotime`: core functions for dates
* :doc:`./pyacs.lib.coordinates`: core functions for coordinates
* :doc:`./pyacs.lib.euler`: core functions for Euler poles
* :doc:`./pyacs.lib.faultslip`: simple core functions for fault slip
* :doc:`./pyacs.lib.glinalg`: core functions for usual linear algebra operations in geodetic problems
* :doc:`./pyacs.lib.gpstime`: functions for gps dates
* :doc:`./pyacs.lib.robustestimators`: a set of linear robust estimators
* :doc:`./pyacs.lib.shapefile`: functions for shapefiles conversions. Useful to visualize results with QGis
* :doc:`./pyacs.lib.strain`: strain rate linear estimation
* :doc:`./pyacs.lib.units`: a few unit conversions functions
* :doc:`./pyacs.lib.utils`: some useful functions for pyacs developpers
* :doc:`./pyacs.lib.vel_field`: horizontal velocity field analysis (Euler pole, profiles, strain rate)
