pyacs core librariesÂ¶

  • ./pyacs.lib.astrotime: core functions for dates

  • ./pyacs.lib.coordinates: core functions for coordinates

  • ./pyacs.lib.euler: core functions for Euler poles

  • pyacs.lib.faultslip package: simple core functions for fault slip

  • ./pyacs.lib.glinalg: core functions for usual linear algebra operations in geodetic problems

  • ./pyacs.lib.gpstime: functions for gps dates

  • ./pyacs.lib.robustestimators: a set of linear robust estimators

  • ./pyacs.lib.shapefile: functions for shapefiles conversions. Useful to visualize results with QGis

  • ./pyacs.lib.strain: strain rate linear estimation

  • ./pyacs.lib.units: a few unit conversions functions

  • ./pyacs.lib.utils: some useful functions for pyacs developpers

  • ./pyacs.lib.vel_field: horizontal velocity field analysis (Euler pole, profiles, strain rate)
